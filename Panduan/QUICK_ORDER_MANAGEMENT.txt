🎯 FITUR TERSEDIA:

1. 👁️  VIEW    - Lihat detail pesanan lengkap
2. ✏️  EDIT    - Edit status & alamat pengiriman  
3. 🗑️  DELETE  - Hapus pesanan (PERMANEN!)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 CARA MENGGUNAKAN:

STEP 1: Login Admin
   → http://localhost/project/admin/login.php

STEP 2: Buka Management Pesanan
   → http://localhost/project/admin/orders.php

STEP 3: Pilih Aksi

   📌 LIHAT DETAIL (👁️)
      → Klik tombol biru (icon mata)
      → Modal muncul dengan detail lengkap
      → Lihat data customer, alamat, produk, total
   
   📌 EDIT PESANAN (✏️)
      → Klik tombol kuning (icon edit)
      → Modal form edit muncul
      → Edit:
         • Status Pesanan (pending → delivered)
         • Status Pembayaran (pending → paid)
         • Status Pengiriman (pending → shipped)
         • Alamat pengiriman
         • Catatan pesanan
      → Klik "Simpan Perubahan"
      → Data ter-update! ✅
   
   📌 HAPUS PESANAN (🗑️)
      → Klik tombol merah (icon trash)
      → Konfirmasi dialog muncul
      → Klik OK → Pesanan TERHAPUS PERMANEN! ❌

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📊 STATUS YANG BISA DIUBAH:

Status Pesanan:
  • pending      → Menunggu konfirmasi
  • confirmed    → Dikonfirmasi
  • processing   → Sedang diproses
  • shipped      → Sudah dikirim
  • delivered    → Sudah sampai ✅
  • cancelled    → Dibatalkan
  • refunded     → Di-refund

Status Pembayaran:
  • pending      → Belum bayar
  • paid         → Sudah bayar ✅
  • failed       → Gagal
  • refunded     → Di-refund

Status Pengiriman:
  • pending      → Menunggu
  • preparing    → Sedang disiapkan
  • shipped      → Dikirim
  • delivered    → Terkirim ✅
  • returned     → Dikembalikan

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎬 USE CASE: Customer Sudah Bayar

1. Customer upload bukti transfer
2. Admin buka orders.php
3. Klik Edit (✏️) pada pesanan tersebut
4. Ubah:
   Status Pembayaran → paid
   Status Pesanan → confirmed
5. Klik "Simpan"
6. DONE! Customer dapat notifikasi ✅

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎬 USE CASE: Kirim Barang

1. Barang sudah dikemas & ada resi
2. Admin edit pesanan
3. Ubah:
   Status Pesanan → shipped
   Status Pengiriman → shipped
4. Tambah catatan: "Resi: JNE123456789"
5. Simpan
6. DONE! Customer tahu barang dikirim ✅

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎬 USE CASE: Update Alamat

1. Customer kontak untuk ganti alamat
2. Admin edit pesanan
3. Scroll ke "Alamat Pengiriman"
4. Update:
   - Alamat
   - Kota
   - Kode Pos
5. Simpan
6. DONE! Barang dikirim ke alamat baru ✅

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎬 USE CASE: Cancel Order

REKOMENDASI: Jangan hapus, tapi ubah status!

1. Admin edit pesanan
2. Ubah:
   Status Pesanan → cancelled
   Status Pembayaran → refunded (jika sudah bayar)
3. Tambah catatan alasan cancel
4. Simpan
5. DONE! Order dibatalkan (data tetap ada untuk laporan) ✅

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ PERINGATAN DELETE!

DELETE = HAPUS PERMANEN!

Yang ikut terhapus:
  ❌ Data order
  ❌ Order items (produk yang dipesan)
  ❌ Alamat pengiriman
  ❌ Data pembayaran
  ❌ TIDAK BISA DI-UNDO!

Kapan boleh delete?
  ✅ Order test/dummy
  ✅ Order duplikat
  ✅ Order error

Kapan JANGAN delete?
  ❌ Order yang sudah delivered
  ❌ Order untuk cancel (pakai status "cancelled")
  ❌ Order dengan pembayaran (pakai status "refunded")

2. Sort by tanggal untuk lihat pesanan terbaru
3. Export data sebelum mass delete
4. Backup database secara berkala
5. Tambahkan catatan saat update status (untuk tracking)

🔧 TROUBLESHOOTING:

❌ Modal tidak muncul?
   → Buka browser console (F12)
   → Check error di console
   → Check file ajax/ ada dan bisa diakses

❌ Data tidak ter-update?
   → Check connection database
   → Check permission file
   → Check activity logs untuk error

❌ Delete tidak berfungsi?
   → Allow pop-up di browser
   → Check JavaScript enabled
   → Test dengan browser lain
